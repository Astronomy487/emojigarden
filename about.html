🏵EmojiGarden

🏵EmojiGarden is a simulator sandbox game where you can construct your own virtual garden using emojified cellular automata. you can close the tab and return hours later to see that your tomatoes have grown, your palm trees have dropped coconuts, and your sheep have eaten all your grass.

it is inspired by hay day, ncase's emoji simulator, and @emoji_garden on twitter.

my design goals include cuteness, minimalism, and positive vibes. i want the game to be playable without looking at this page (ಠ_ಠ) and i want there to be no non-numeral non-user text.

how to play

your garden consists of fields, which you can drag around like windows using their grabber.

you can click a cell with an object to act on it. if a cell type has several actions, a dropdown will appear (also accessible by right click). actions include spreading a tile like a flower, dropping other tiles like a tree with acorns, harvesting a crop like a tomato, or removing an object altogether. all objects have at least one remove action.

you can shift click an object to temporarily carry it, and click to set it somewhere new. this is not equivalent to storing into an inventory because of logistics with the cooldown. while held, an object can be deleted (harvested for resources if cooldown allows, otherwise just deleted). alt+shift+click will quickly perform this deletion process too.

🏵EmojiGarden does not use cookies, and i (astro) (the guy who made this) can't see any of your data. your file is saved using localStorage, meaning your browser remembers a small text file that only 🏵EmojiGarden can access.
